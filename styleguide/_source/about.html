About

The Co-op Style Guide forms the basis of the visual & code style for digital projects. It is a work in progress and is based on the principals of Atomic Design.

The Style Guide referrences all our common patterns so far. We have started out small, with what we know now. We will be adding to the guide all the time as we design, build and test new elements. It will constantly change and involve based on real user behaviour.

The source files sit in one centrally located github repository and are intended to serve all digital projects. If you for your developers require access to the repo for use in your project please contact Matt Tyas. More technical information about the system is located in the repository's README.md.
