Design patterns

The Co-op Front-end Toolkit provides core styles and components. The following provides guidance on how to use these within your service or application.

Colours

Colours used for text, links, backgrounds, buttons and statuses.

Typography

Fonts, typographic scale and common text styles.

Icons

Available icons and icon usage.

Links and buttons

Links and buttons styles and usage.

Elements

Styling for common HTML elements (lists, tables, etc).

Layout

Grid system and layout options.

Forms

Styles for fields, buttons and form wrappers.

Components

Visual and functional components.

Templates

A set of templates for base markup and common page elements.
