Typographic scale Back to top

This is the basic typographic scale for use across desktop and mobile sizes. In this basic scale only the H1 is affected by the device size and drops to be more appropriate on a small device.

It may be appropriate to use smaller sizes at mobile to allow for more content to be on screen. This is fine, and at the designers discretion. All we ask is that colour contrast ratios and all other appropriate accessibility critera from our digital guidelines are met.

CSS classes are provided so that should a developer need to create a certain visual hierarchy, they can, without breaking the semantics of the page. This is known as a Double strand heading hierarchy.

H1 – 36px heading (30px mobile)

H2 – 21px heading

H3 – 19px heading

H4 – 16px heading

H5 – 13px heading
H6 – 11px heading

16px should be used for body text. All copy below 18px must be in dark grey (#6e6e6e) for accessibility purposes

Small print should be 13px desktop, 11px mobile

<h1>H1 &ndash; 36px heading (30px mobile)</h1>
<h2>H2 &ndash; 21px heading</h2>
<h3>H3 &ndash; 19px heading</h3>
<h4>H4 &ndash; 16px heading</h4>
<h5>H5 &ndash; 13px heading</h5>
<h6>H6 &ndash; 11px heading</h6>
<p>16px should be used for body text. All copy below 18px must be in dark grey (#6e6e6e) for accessibility purposes</p>
<p class="smallprint">Small print should be 13px desktop, 11px mobile</p>
