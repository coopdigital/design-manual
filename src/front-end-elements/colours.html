Brand colour Back to top

Our brand blue should only be used for the logo. It should never be used as text or as a background colour. This is becuase it has poor contrast and may be difficult for those with low vision to read on screen.

Text colours Back to top

Text colours meet web accessibility standards that the WCAG 2.0 specify for text and background color combinations. We chose the $coop-blue as a safer alternative to $coop-brand-blue. It must be used sparingly and only for text (not background colour) above 18px to achieve a 3.1 contrast ratio. The colour contrast we look for on body text is 4.5:1 and3.1 on large text (above 18px or 14px bold).

If you are unsure about any of the colours you are using below are some tools that can help you to check and test them.

  • Web Aim's Color Contrast Checker
  • La Verou's Colour Contrast Analyser
  • Color Oracle (free color blindness simulator)

Ultimately, there is no subsitiute for testing with real people of all abilities to see if your users can interact with your service.
