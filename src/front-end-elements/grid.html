Sizes

The default Grid provides classes spanning 12 columns, and provides breakpoints at the following screen sizes:

  • All - 0px and up
  • Extra small - 320px and up
  • Small - 480px and up
  • Medium - 750px and up
  • Large - 960px and up
  • Extra large - 1300px and up

These settings can be overriden by specifying your own list of screen sizes you want columns available for. To keep the size of the resulting production stylesheet down, it is recommended to taylor the settings to your requirements, to make sure only the necessary classes are generated.

Layout helpers

The following helpers are included by default in the resulting CSS. If you do not make use of them, it is recommended to exclude them by setting them to false:

These helpers can be applied to specific breakpoints only.

$grid-include-center-classes: false;
    $grid-include-pull-classes: false;
    $grid-include-push-classes: false;
    $grid-include-offset-classes: false;

Centering classes

These allow centering of columns which do not fill the whole width of the viewport.

Example centering a block

.all-6 .all-centered

The centering can be undone for subsequent breakpoints:

Example Centering and uncentering blocks across sizes

.all-6 .all-centered .large-uncentered
.all-6 .all-centered .large-uncentered

Offsetting classes

These classes allow adding an offset to the left of any column. They can be applied to specific breakpoints only.

Example Offsetting blocks on the grid on certain sizes

.small-4 .small-offset-1 .large-6 .large-offset-0
.small-4 .small-offset-2 .large-offset-1

Push/pull classes

These classes allow re-ordering of the columns by pushing and pulling them relative to their base position.

Example Re-ordering blocks across sizes

.small-6 .small-push-6 (first in markup)
.small-6 .small-pull-6 (last in markup)
.small-4 .small-push-4 (first in markup)
.small-4 .small-push-4 (second in markup)
.small-4 .small-pull-8 (last in markup)

Collapse columns

Collapsed columns have their padding removed. Columns can be collapsed individually, or whole rows can be collapsed at once by adding the class .collapse to the relevant column or row element.

Example Removing column gutters

.all-6 .collapse
.all-6
.row.collapse > .all-6
.row.collapse > .all-6
