Links

A link should be descriptive of the place it's taking the user. It should also reflect the title (if possible) of the destination page, so the user knows they've arrived at the right place. Avoid using text such as, 'click here', or anything that does not make sense when read out of context. Screen readers for instance, read link text in isolation without the surrounding content.

Buttons

Buttons are made using Avenir Next Medium at 20px (1em), a 3px border radius, 15px 37.5px padding and a box-shadow. The background colour is lightened on hover. Buttons take a 4px blue outline on focus and active.

For desktop and tablet formats button width should be defined by the button content. Buttons should be ranged left in these formats; the left edge aligning with other content.
